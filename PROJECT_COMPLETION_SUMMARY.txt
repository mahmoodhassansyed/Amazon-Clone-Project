   - Displays all order items

   - No backend required - fully client-side

================================================================================
DATA PERSISTENCE
================================================================================

SessionStorage Data:
- 'cart': Current shopping cart items (array)
- 'orders': All placed orders (array)
- 'selectedOrderIndex': Currently viewing order (tracking page)

Persistence Behavior:
- Data persists during browser session
- Data clears when browser closes
- Data survives page reloads
- Each browser tab has separate storage

Data Structure:
Cart Item:
{
  productId: string,
  quantity: number,
  deliveryDate: string (formatted date)
}

Order:
{
  orderHeader: {
    dateOrdered: string (YYYY-MM-DD),
    total: number (dollars),
    orderId: string (unique)
  },
  orderDetails: [
    {
      dateArrival: string,
      image: string (path),
      name: string,
      quantity: number
    }
  ]
}
